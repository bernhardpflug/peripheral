Binary - This package contains the pre-compiled Java jar file needed to run the example measurement probe application. Download this package if you want to run the application straight out of the box. You will need the source package if you want to make changes in the code later.

JddacClient-0.5-bin.zip (Approximately 0.2 MB)

Source - This package contains the source code and the documentation, including the Javadocs. Download this package if you would prefer to compile the source code yourself or if you would like your own copy of the source code installation.

jddac-0.5-src.zip (Approximately 5 MB)
