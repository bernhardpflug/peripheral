June 28, 2004

JDDAC Release Notes Version 0.x

The Entazza Crew

Overview

This note describes the releases of the JDDAC package. Release 0.1 includes elements of the JMDI and JTI APIs for the J2SE platform.   In release 0.2 J2ME platform support was added.

The purpose of this release is to provide an early peek at the JDDAC framework.The release contains a customizable measurement application framework built on JMDI and JTI as well as an simple example application. This example application makes measurements and reports the measurements to a measurement server.

Based on the examples in this sample application, a Java developer can modify it to make other types of measurements or perform other functions.

The measurement server  is running on a J2EE version of JDDAC. The source code for the J2EE version will be released in the near future. For running the code there is a measurement portal server running at Agilent Laboratories in Palo Alto, California open to the public.  

The web pages on the measurement server has been tested with Microsoft Internet Explorer 5.5. There may be problems with other browsers as there use of Javascript in the pages. If you see problems with your browser, please post a message at the JDDAC discussion forum on java.net and we will try to help you with it.
